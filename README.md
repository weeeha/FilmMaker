# FilmMaker

AI-powered film storyboard planner. Paste a screenplay, the app extracts scenes and shots, then auto-generates a visual reference image for each shot (with style propagation across the film).

Phase 1: create projects, upload screenplays, generate AI images per shot. Drag-to-reorder and PDF export planned next.

## Stack

- Next.js 16 + AI SDK v6
- Vercel AI Gateway (gpt-image-1)
- Tailwind v4 + shadcn/ui
- localStorage persistence
